# 📚 WordMaster — 개인용 영단어 학습 툴

오직 빠르고 확실한 단어 암기를 위해 직접 만든 웹 기반 영단어장입니다. 복잡한 설치나 서버 없이, 브라우저만 있으면 언제 어디서든 즉시 학습할 수 있도록 단일 HTML 파일로 가볍게 제작했습니다. 

치열한 수능 준비 스케줄 속에서, 자투리 시간을 100% 활용하고 극한의 난이도까지 완벽하게 대비하기 위해 필요한 모든 기능을 눌러 담았습니다.

## ✨ 주요 기능 (Key Features)

* **🧠 간격 반복 학습 (SRS, SM-2 알고리즘)**
  * 망각 곡선에 기반하여 단어별 복습 주기를 자동으로 계산합니다. 
  * 매일 꼭 복습해야 할 단어만 효율적으로 학습할 수 있습니다.
* **🃏 다기능 플래시카드 & 퀴즈**
  * 앞뒤로 뒤집어보는 플래시카드 (자동 넘김, 배속 설정 지원)
  * 객관식 뜻 맞히기 및 주관식 타임어택 퀴즈
  * 오타 허용 수준(엄격/보통/느슨) 설정 및 유사어 정답 인정 기능
* **🔥 어려운 뜻(심화) 모드 & 숙어장**
  * 다의어 처리를 위해 단어마다 '어려운 뜻'을 별도로 지정하여 심화 학습이 가능합니다.
  * 단어장과 완전히 분리된 전용 '숙어장'을 지원합니다.
* **📋 스마트 관리 및 오답 노트**
  * 틀린 단어만 자동으로 모아주는 오답 노트
  * 드래그 앤 드롭으로 직관적인 단어 난이도/완료 상태 분류 (칸반 보드 스타일)
  * 다중 태그 시스템 및 일괄 태그 추가 기능
* **💾 완전한 로컬 데이터 처리**
  * IndexedDB와 localStorage를 활용하여 모든 데이터가 브라우저에 안전하게 저장됩니다.
  * JSON 파일 형태로 데이터를 내보내고(백업) 불러올 수 있습니다.

## 🛠 사용 기술 (Tech Stack)

* **Frontend:** Vanilla HTML, CSS, JavaScript (No Frameworks)
* **Storage:** LocalForage (IndexedDB with LocalStorage fallback)
* **Design:** 커스텀 다크/라이트 모드 테마, 반응형 UI 설계

## 🚀 시작하기 (How to Use)

별도의 서버나 데이터베이스 세팅이 필요하지 않습니다.

1. 이 저장소의 `wordmaster.html` 파일을 다운로드합니다.
2. 크롬, 사파리 등 웹 브라우저로 해당 파일을 엽니다.
3. 바로 단어를 추가하고 학습을 시작하세요!
*(GitHub Pages가 활성화되어 있다면 제공된 링크를 통해 모바일이나 태블릿에서도 접속할 수 있습니다.)*

## 💡 개발 배경

이 프로젝트는 저의 첫 번째 개발 작품입니다. 시중에 있는 단어장 앱들이 채워주지 못하는 가려운 부분들(예: 극악의 난이도 대비, 내 맘대로 되는 태그 필터, 가벼운 구동 등)을 직접 해결해 보고자 만들었습니다.
